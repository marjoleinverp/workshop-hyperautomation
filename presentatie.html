Wat is Hyperautomation?

2 / 9

De evolutie

  • Automation: Eenvoudige taken automatiseren (macro's, scripts)
  • Workflow Automation: Systemen aan elkaar koppelen (n8n, Power Automate)
  • Hyperautomation: AI + automation = slimme beslissingen

Het grote verschil

Traditionele automation volgt regels.

Hyperautomation denkt mee.

AI leest, begrijpt, classificeert en handelt - ook bij ongestructureerde input.

n8n: Ons Platform

4 / 9

Waarom n8n?

  • Visueel: Blokjes slepen, verbinden, klaar
  • Open source: Draait op onze eigen server
  • 400+ integraties: Van Outlook tot Kadaster API
  • AI-ready: Directe koppeling met ChatGPT, Claude
  • Geen vendor lock-in: Jouw data, jouw controle

Voorbeeld 1: Slimme Topdesk Intake

5 / 9

De situatie

Meldingen komen binnen via mail, formulier, of telefoon. Iemand moet handmatig:

  • De melding lezen en begrijpen
  • Categorie bepalen
  • Prioriteit inschatten
  • Toewijzen aan juiste team

De oplossing

Webhook
->
GPT Analyse
->
Topdesk API

Resultaat

AI leest de melding en bepaalt:

  • Categorie: "Openbare ruimte > Groen"
  • Prioriteit: P3 (standaard)
  • Team: Buitendienst Veghel
  • Samenvatting: "Overhangende tak bij speeltuin"

Bonus: P1/P2? Direct Teams-alert!

Voorbeeld 2: Automatische Adresverrijking

6 / 9

De situatie

Bij een nieuwe zaak/aanvraag moet je handmatig:

  • Adres opzoeken in BAG
  • Eigenaar checken in Kadaster
  • Bestemmingsplan raadplegen
  • Alles samenvoegen in dossier

De oplossing

Adres input
->
BAG API
->
Kadaster
->
Document

Resultaat

In 3 seconden een compleet dossier:

  • Officieel BAG-adres + coordinaten
  • Bouwjaar, oppervlakte, bestemming
  • Eigendomsgegevens
  • Relevante vergunningshistorie

Tijdsbesparing: 15 min -> 3 sec
